## 1. 端口映射
    `docker run -itd -p 宿主机端口号:容器端口号`

## 2. 文件挂载
    `docker run -itd -v /宿主机/文件目录/文件名:/容器/目录/文件名`

  #### 1. 将容器的文件复制到宿主机
        `docker cp /容器名:/容器目录/文件名 /宿主机目录/文件名`
    
  #### 2. 将宿主机的文件复制到容器
        `docker cp /宿主机目录/文件名 /容器名:/容器目录/文件名`

## 3. 容器互联
    `docker run -itd --link 要关联的容器名称：容器在被关联的容器中的别名`
